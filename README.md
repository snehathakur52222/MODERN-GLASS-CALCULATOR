# 🧮 Modern Glass Calculator

A modern and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project features a premium glassmorphism user interface, animated gradient background, keyboard support, and smooth button interactions for an enhanced user experience.

## 📌 Introduction

The Modern Glass Calculator is a responsive web application that performs basic arithmetic operations with a sleek and interactive design. It combines HTML for structure, CSS for modern styling and animations, and JavaScript for handling calculations and user interactions. This project demonstrates core frontend development skills and responsive UI design.

---

## ✨ Features

- 🧮 Perform basic arithmetic operations (+, −, ×, ÷)
- 📊 Percentage (%) calculation
- ⌫ Delete last entered character
- ❌ Clear display
- ⌨️ Keyboard support
- 💎 Modern Glassmorphism UI
- 🌈 Animated gradient background
- ✨ Smooth hover and click animations
- 📱 Fully responsive design
- ⚠️ Error handling for invalid expressions

---
